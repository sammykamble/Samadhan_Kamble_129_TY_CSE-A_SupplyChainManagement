Query 4 - Supplier Performance:
SELECT s.supplier_name, COUNT(po.order_id) as orders, 
       SUM(po.total_amount) as total_value
FROM suppliers s
LEFT JOIN purchase_orders po ON s.supplier_id = po.supplier_id
GROUP BY s.supplier_id
ORDER BY total_value DESC;

Query 5 - Warehouse Utilization:

================================================================================
                        TROUBLESHOOTING
================================================================================

PROBLEM 1: Apache Won't Start - Port 80 Already in Use
-------------------------------------------------------
SOLUTION:
1. Open XAMPP Control Panel
2. Click "Config" button next to Apache
3. Select "Apache (httpd.conf)"
4. Find line: Listen 80
5. Change to: Listen 8080
6. Save file and close
7. Click "Config" again â†’ "Apache (httpd-ssl.conf)"
8. Find line: Listen 443
9. Change to: Listen 4433
10. Save and restart Apache
11. Access application via: http://localhost:8080/supplychain/

PROBLEM 2: MySQL Won't Start - Port 3306 Already in Use
--------------------------------------------------------
SOLUTION:
1. Open XAMPP Control Panel
2. Click "Config" button next to MySQL
3. Select "my.ini"
4. Find line: port=3306
5. Change to: port=3307
6. Save file and close
7. Restart MySQL
8. Update database connection:
   - Open: C:\xampp\htdocs\supplychain\config\database.php
   - Change: define('DB_SERVER', 'localhost');
   - To: define('DB_SERVER', 'localhost:3307');
   - Save file

PROBLEM 3: "Access Denied for user 'root'@'localhost'"
-------------------------------------------------------
SOLUTION:
1. Open phpMyAdmin
2. Click "User accounts" tab
3. Find user "root" with host "localhost"
4. Click "Edit privileges"
5. Click "Change password"
6. Set a password or select "No password"
7. Click "Go"
8. Update config/database.php:
   - Change: define('DB_PASSWORD', '');
   - To: define('DB_PASSWORD', 'your_password');

PROBLEM 4: "Table 'supply_chain_db.suppliers' doesn't exist"
-------------------------------------------------------------
SOLUTION:
1. Database not created properly
2. Go to phpMyAdmin
3. Run COMPLETE_SQL_FILE.sql again
4. Verify all tables are created
5. Refresh application

PROBLEM 5: CSS Not Loading / Page Looks Broken
-----------------------------------------------
SOLUTION:
1. Check folder structure is correct
2. Verify files are in: C:\xampp\htdocs\supplychain\
3. Clear browser cache (Ctrl + Shift + Delete)
4. Hard refresh page (Ctrl + F5)
5. Check browser console for errors (F12)

PROBLEM 6: "Cannot modify header information"
----------------------------------------------
SOLUTION:
1. Open the PHP file showing error
2. Ensure no whitespace before <?php tag
3. Ensure no whitespace after ?> tag
4. Save file with UTF-8 encoding (no BOM)

PROBLEM 7: phpMyAdmin Not Accessible
-------------------------------------
SOLUTION:
1. Ensure MySQL is running in XAMPP
2. Try: http://127.0.0.1/phpmyadmin
3. Check httpd-xampp.conf for phpMyAdmin alias
4. Restart Apache service

PROBLEM 8: "Call to undefined function mysqli_connect()"
---------------------------------------------------------
SOLUTION:
1. Open php.ini file (XAMPP Control â†’ Apache Config â†’ php.ini)
2. Find line: ;extension=mysqli
3. Remove semicolon: extension=mysqli
4. Save file
5. Restart Apache

PROBLEM 9: Session Not Working / Auto Logout
---------------------------------------------
SOLUTION:
1. Check session.save_path in php.ini
2. Ensure C:\xampp\tmp folder exists
3. Give write permissions to tmp folder
4. Restart Apache

PROBLEM 10: Stored Procedures Not Working
------------------------------------------
SOLUTION:
1. In phpMyAdmin, select supply_chain_db
2. Click "Routines" tab
3. Verify procedures are listed
4. If not, run sql_queries.sql again
5. Ensure DELIMITER is properly set
